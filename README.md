# ⏱️ Stopwatch Web Application

A responsive and interactive stopwatch web app built with **HTML**, **CSS**, and **JavaScript** — featuring start, pause, reset, and lap time tracking.

---

## 📸 Preview

![Stopwatch App Screenshot](Screenshot_2026-04-02_165858.png)

---

## 🚀 Features

- ▶️ **Start / Pause** — Control the timer with a single button
- 🔄 **Reset** — Reset the stopwatch back to zero
- 🏁 **Add Lap** — Record lap times while the stopwatch is running
- 🗑️ **Clear All** — Clear all recorded lap times at once
- 📱 **Responsive Design** — Works seamlessly across all devices

---

## 🛠️ Built With

- **HTML** — Structure of the application
- **CSS** — Styling and responsive layout
- **JavaScript** — Timer logic, lap tracking, and interactivity
